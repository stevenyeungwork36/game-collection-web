# Playing card images for Big Two (鋤大弟)

Place your card **PNG** (or **JPG**) files in this folder. The game will use them when present; otherwise it shows a text fallback.

## File naming

One file per card: **`{rank}_of_{suit}.png`**

- **Rank:** `2`, `3`, `4`, `5`, `6`, `7`, `8`, `9`, `10`, `ace`, `jack`, `queen`, `king`
- **Suit:** `clubs`, `diamonds`, `hearts`, `spades` (all lowercase)

## Examples

| Card    | Filename              |
|---------|------------------------|
| 3♦      | `3_of_diamonds.png`   |
| 10♥     | `10_of_hearts.png`    |
| Jack ♣  | `jack_of_clubs.png`   |
| Ace ♠   | `ace_of_spades.png`   |
| 2♦      | `2_of_diamonds.png`   |

## Total files

52 cards = 13 ranks × 4 suits.  
Recommended size: about **140×200 px** (or similar 2:3 ratio) so cards look sharp and consistent.
